## Features

* 12-hour time with AM/PM
* Live date
* Soft fade + blinking colon
* Minimal grayscale color picker
* OLED-friendly pure black background
* Fully responsive
* No dependencies

## Usage

Open `index.html` in a browser or deploy to Vercel.
